배포 전 체크리스트:
  1) config.authUrl / config.keycloakIssuer 를 실제 외부 URL로 설정
  2) 비밀값: secrets.authSecret(openssl rand -base64 32) / secrets.databaseUrl / secrets.keycloakSecret
     (또는 secrets.existingSecret 로 외부 관리 Secret 참조)
  3) 외부 Keycloak realm(simple-wiki)·client(simple-wiki-app)의 redirectUris 에 authUrl 등록
{{- if gt (int .Values.app.replicaCount) 1 }}
  4) 주의: app.replicaCount={{ .Values.app.replicaCount }} 인데 persistence.accessMode={{ .Values.persistence.accessMode }} 입니다.
     첨부파일을 여러 파드가 공유하려면 ReadWriteMany 볼륨이 필요합니다(디스크 저장 한계).
{{- end }}
